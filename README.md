# LED-Dice-74HCT4060

 LED Spielwürfel (Dice) mit dem 14-Bit Zähler 74HCT4060

## Funktionsbeschreibung

 * Der Würfel zeigt beim Drücken des Tasters die Zahlen von 1-6 an.
 * Solange der Taster SW1 nicht gedrückt ist leuchten alle LED's
